# 🚗 Vehicle Price Prediction System

A full-stack Vehicle Price Prediction web application built using **Python Flask**, **Machine Learning**, **SQLite**, **HTML**, **CSS**, and **JavaScript**.

Users can:
- Register and Login
- Predict vehicle resale price using ML
- Upload vehicle images
- View prediction history
- Access dashboard analytics
- Admin panel support

# ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/vehicle-price-prediction.git
```

Open project:

```bash
cd vehicle-price-prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python app.py
```

Open browser:

```plaintext
http://127.0.0.1:5000
```
